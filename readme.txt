== Description ==

Czech: Jednoduchý plugin, který přidává widget co zobrazuje aktuální teplotu nebo vlhkost naměřené čidly na www.tmep.cz.

Přímo ve widgetu jde nastavit, které informace se mají zobrazit.

English: Simple plugin whats add widget with temperature and humidity from www.tmep.cz.

== Installation ==

Czech:
1. Nahrajte kompletní adresář pluginu do `/wp-content/plugins/` nebo instalujte plugin obvyklým způsobem z repozitáře wordpress.org
1. Aktivujte plugin 'TMEP.cz widget' v administraci plug-inů
1. Přiedejte 'TMEP.cz widget' do zvolené oblasti (např. postranního panelu)
1. Nastavte widget podle postupu na obrázcích

English:
1. Upload the plugin files to the `/wp-content/plugins/tmep-cz` directory, or install the plugin through the WordPress plugins screen directly
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Add the 'TMEP.cz widget' to your widget area
1. Set the widget
